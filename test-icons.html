❌ 问题描述

原因：Element Plus Icons 中不存在 ExternalLink 图标。

已修复：将 ExternalLink 替换为 TopRight

修改内容：

❌ ExternalLink
✅ TopRight

👁️ View - 查看按钮

📋 DocumentCopy - 复制按钮
✅ Check - 复制成功状态
↗️ TopRight - 打开链接按钮

📝 修复详情

1. 导入语句修复：

修改前：
import { View, Link, DocumentCopy, Check, ExternalLink } from '@element-plus/icons-vue'

修改后：
import { View, Link, DocumentCopy, Check, TopRight } from '@element-plus/icons-vue'

2. 模板使用修复：

修改前：
<el-icon><ExternalLink /></el-icon>

修改后：
<el-icon><TopRight /></el-icon>

3. 复制按钮图标优化：

修改前：
:icon="copySuccess ? 'Check' : 'DocumentCopy'"

修改后：
<el-icon>
  <Check v-if="copySuccess" />
  <DocumentCopy v-else />
</el-icon>
